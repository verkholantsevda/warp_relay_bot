# warp_relay_bot# Warp Relay Bot

Telegram бот для генерации персональных конфигураций **Cloudflare WARP / WireGuard**.

Бот автоматически создаёт ключ доступа и выдаёт готовый `.conf` файл для подключения к защищённому интернет-каналу.

👉 Попробовать бота:  
https://t.me/warp_relaybot

---

## Возможности

- 🌍 Генерация WARP ключей для разных стран
- 🔑 Персональные ключи для каждого пользователя
- 📄 Просмотр ранее созданных ключей
- 📥 Скачать конфигурацию `.conf`
- ❌ Удаление старых ключей
- 📱 Быстрые ссылки на мобильные приложения

---

## Как это работает

1. Пользователь открывает Telegram бота
2. Выбирает страну сервера
3. Бот генерирует конфигурацию WireGuard
4. Конфигурация сохраняется в базе
5. Пользователь может скачать `.conf` или посмотреть его

---

## Ограничения

Для предотвращения злоупотреблений действует лимит:

- максимум **5 ключей на пользователя**

Старые ключи можно удалить в разделе **"Мои ключи"**.
